<span>HOW TO COMPILE (LINUX) </span><br>
<span>- - - - - - - - - - - - - - - - - - - - -<br></span>

To compile, configure and install dbuster, just type:<br>
    
If you're in sudo mode or rooted, just type:<br>
    
```powershell
chmod +x configure.sh
```
    
```
./configure.sh -r
```

or    

```
./configure.sh --root
```
    
if you're a termux user, just type:<br>

```
./configure.sh -t
```
    

```
./configure.sh --termux
```
